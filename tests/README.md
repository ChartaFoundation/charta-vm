# Charta Tests

Test suites for Charta implementation.

## Test Structure

- **Unit tests** - Individual component testing
- **Integration tests** - Component interaction testing
- **Governance tests** - Safety and governance pattern testing
- **End-to-end tests** - Complete program execution testing

## Test Categories

### Core Semantics Tests

- Signal/coil store management
- Rung evaluation
- Guard evaluation (NO/NC, AND/OR)
- Cycle execution
- Timer semantics

### Safety Tests

- Fail-safe gate patterns
- Governance interlock patterns
- Safety-critical path verification
- Effect system constraints

### Governance Tests

- Explainability requirements
- Contestability flows
- HITL enforcement
- Appeal window management

### Integration Tests

- Block evaluation
- Network execution
- Agent coordination
- MCP tool integration
- Rules-as-code integration
